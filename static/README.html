ASCII ART WEB

This project displays an html page where you can put text, select a police and, once you submit, the text you entered will be displayed in graphic representation using ASCII. To launch the project you need to execute the file “server.go” that launches a server, then open a web browser, go to URL page http://localhost:8080/ where you can test the project.
